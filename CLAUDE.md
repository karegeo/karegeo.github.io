# karegeo.github.io

Personal landing page hosted via GitHub Pages.

## Conventions for changes

When editing `index.html` (or any page on the site):

1. **Update the "Last updated" date** in the footer to today's date
   (YYYY-MM-DD). This is a manual marker so the author can confirm a
   deployment landed and visitors can see when the page changed.
2. **Keep the Claude Code disclaimer** in the footer
   ("Built with the help of [Claude Code](https://claude.com/claude-code).").
